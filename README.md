# Learning Application - Frontend POC

This project demonstrates the frontend features of a learning application.

## Features Implemented

- YouTube video player using embed
- Course playlist UI
- Thumbnail + video title
- Progress tracking
- Autoplay next video

## Tech Stack

React Native
Expo
YouTube Embed Player

## UI Flow

Home Screen
↓
Course Player Screen
↓
Video Player + Playlist
↓
Progress Tracker

## Demo

Run using:

npx expo start
